# Setup Development Environment

This command sets up your complete development environment:

1. Generates AI agent wallet
2. Creates .env from example
3. Installs dependencies
4. Verifies Redis connection

```bash
npm run setup
```

After setup:
- Fund agent wallet (address shown in output)
- Run: npm run demo:siwx
- Start MCP server: npm run dev
